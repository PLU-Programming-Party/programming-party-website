# ProgrammingPartyWebsite - Design Artifacts

This directory contains RAPID method design artifacts organized by phase:

| Directory | Phase | Contents |
|-----------|-------|----------|
| `1-domain/` | R (Requirements) | Domain Specifications (DS-xx) |
| `2-logical/` | A (Architecture) | Logical Designs (LD-xx) |
| `3-planning/` | P (Planning) | Implementation Designs (ID-xx) |
| `4-implementation/` | I (Implementation) | Delivery artifacts |
| `archive/` | - | Closed iteration archives |

## Getting Started

1. Run `rapid_status.py --ai` to see current project state
2. Use `rapid_lookup.py --search "topic"` to find method guidance
3. Use `rapid_session.py --start` to begin a session
